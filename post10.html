Industrial Hoist Lifting Mechanism

Dynamic System Design Project

This project was started in my sophomore year in college in which I designed a specialized hoist tailored to a unique application.

The task was to recommend based on given specifications and applying engineering calculations, a commercially available AC squirrel cage induction motor to be integrated with already available components.

The major components already available for use were the speed reducer, cable drum and a multi-pulley cable assembly system.

One of eight drum-pulley configurations were given as options to be integrated with the specialized hoist.

At the end, a letter of transmittal was generated in addition to a technical report following ISO 7200 format.

Technologies and Concepts Used:
  • - Dynamics - lifting force, speed
  • - Lifting hoist mechanism
  • - Technical documentation
