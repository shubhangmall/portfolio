Full MERN Stack Expense Tracker

I coded this expense tracker React application using create-react-app, hooks (useState, useContext, useReducer) and context API.

I added a working backend API with Express, MongoDB and Node js.

Finally, I created an npm script to deploy the app on a production build.

This app keeps track of your income and expenses, you are able to add new transactions and it will store it into the MongoDB database. There are red and green color-codes for loss and gains in wealth.

Technologies Used:
  • - React
  • - create-react-app
  • - React hooks - useState, useContext, useReducer
  • - Context API
  • - Node js
  • - MongoDB
  • - Express
  • - VS Code - development and production building
