Current Weather App

I coded this weather app that uses Vanilla JavaScript mainly, it allows you to search for and enter a city name in the search bar.

Afterwards, it will give you the current temperature in Celsius, a phrase describing the weather, and the high/low temperature of the desired city. There is also the current date.

Overall, the UI front-end was developed using HTML and CSS.

It then fetches data to an external weather API and returns the required data to display. There are live updates to the data.

Technologies Used:
  • - HTML
  • - CSS
  • - Vanilla JavaScript
  • - Open Weather Map API
  • - RESTful API - return data
  • - Modern JavaScript Fetch API
