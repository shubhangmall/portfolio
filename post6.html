Quiz App

During quarantine I started learning front-end development languages. One of the first projects I created was this quiz trivia app.

The home screen contains a play button, login, and sign up buttons. These buttons use event handlers for sound and action. Currently, the play button is the only one I implemeneted to start the quiz.

After clicking on the play button, the instruction page will load that explains the rules of the quiz game and how to play.

I also added a 50-50 lifeline that will remove 2 wrong answers when clicked, leaving one correct and one wrong answer.

To further help the contestant, another lifeline I added was a maximum of 5 hints throughout the entire quiz that removes 1 wrong answer each time clicked.

After reading the instructions and clicking on play, a static array of 15 predetermined trivia questions load up and the game begins. One can quit at any time and go back to the home screen.

Technologies Used:
  • - React
  • - JavaScript
  • - HTML
  • - CSS
  • - React components - Helmet, Link, Fragment, Router
