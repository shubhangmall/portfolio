Model Based Design Pacemaker

This project was started in my junior year of college as part of a Software Development course.

A pacemaker is a safety critical device implanted in the body that is used to monitor and regulate a patient's heart rate by sending electrical pulses to the heart.

The goal was to describe the modes and operations of a cardiac pacemaker using a Model Based Design (MBD) approach. This requires a great need for reliable software specification, verification, validation and quality in today's medical devices. In addition, failure for software to meet safety and security standards can be fatal to the user. Lastly, this approach makes use on reliable Finite State Machine (FSM) based formal methods.

Another goal was to improve familiarity with the abovementioned model-based design as well as going through the software design process from natural language specifications to verification.

The last part was to go through a testing procedure involving hardware test benchmarks and Simulink modelling and debugging.

Technologies and Concepts Used:
  • - Cardiac Pacemaker
  • - Shield - FRDM-K64F
  • - Model-Based Design
  • - Software Design Process
