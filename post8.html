Analog to Digital Conversion: Tracking ADC 4-bit

This project was started in my sophomore year in college and helped teach the importance of Analog to Digital (A to D) conversion.

The tracking ADC is one of the most common A to D converters as it is useful in tracking slow changing signals.

I designed, simulated, implemented and tested a 4-bit tracking ADC using common commercially available circuit elements and components.

The goal was to choose from the available components while meeting design requirements and paying attention to overall project budget.

The simulation phase uses Multisim software taking an analog input in the form of a triangular wave with a peak-to-peak value of 200 mV and DC offset of 100 mV which is outputted as a digital signal using a logic analyzer and tracked using an oscilloscope.

The implementation involved building the designed analog-digital circuit on a breadboard and testing it using a data acquisition platform. The digital output signal was demonstrated using Light Emitted Diodes (LEDs).

Technologies and Concepts Used:
  • - Comparator - dual linear with CMOS, MOS, TTL
  • - Counter - 4-bit synchronous binary
  • - Clock Signal
  • - Digital to Analog Converter (DAC)
  • Passive circuit elements - LEDs, resistors, capacitors, inductors, transformers, etc.
  • - Logic Chips - 74HC00N quad NAND gate
  • - Multisim simulation software
  • - myDAQ data acquisition platform
