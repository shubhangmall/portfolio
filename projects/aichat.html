Artificial Intelligence Career Chatbot

  • Tech Stack: uv, OpenAI, Google Gemini, CrewAI, Ollama, Cursor, HuggingFace, Pushover, Python
  • App URL: Project Link

Created a web‑based AI‑powered career chatbot application that serves as a professional digital persona, capable of answering questions about background, skills, and experience while capturing user interest and feedback through intelligent conversation.

Implemented OpenAI's function calling API with custom tools for lead capture and question tracking; integrated Pushover API for real‑time mobile notifications when users engage, reducing response time for follow‑ups by enabling instant alerts for new inquiries and unknown questions..

Deployed scalable chatbot to HuggingFace Spaces using Gradio framework with PDF resume parsing and dynamic conversation management, enabling 24/7 professional representation and automated lead generation.
